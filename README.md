# Primitive DB - Консольная база данных на основе JSON

## Описание
Primitive DB - это консольное приложение для работы с табличной базой данных,
основанной на файлах JSON. Приложение поддерживает создание, удаление таблиц
и основные операции с данными через интуитивно понятный CLI интерфейс.

## Установка

### Требования
- Python 3.11 или выше
- Poetry 1.2.0 или выше (пакетный менеджер)
- Windows 11 / Linux / macOS
